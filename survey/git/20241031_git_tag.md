# git tag

```sh
# list tag（`*` is allowed）
git tag -l `[keyword]`
# create tag
git tag `[tagName]`
# create with commit
git tag `[tagName]` `[commit|tag]`
# create tag with annotations
git tag -a `[tagName]` -m `[message]`
# delete tag
git tag -d `[tagName]`
# show tag informaton
git show `[tagName]`
```
